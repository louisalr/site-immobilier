## What you need to run this project
    
    PHP
    PostgreSQL
    Nodejs
    
## How to run this project

    Edit the .env file with your credentials
    
    DB_CONNECTION=pgsql
    DB_HOST='IP'
    DB_PORT='port' 
    DB_DATABASE='name of database'
    DB_USERNAME='name'
    DB_PASSWORD='password'
    
Then 

    php artisan serve
